Ստեկ (Stack)

Ստեկը (Stack) գծային տվյալների կառուցվածք է, որն աշխատում է LIFO (Last In, First Out) սկզբունքով։ Սա նշանակում է, որ վերջին ավելացված («push» եղած) տարրը առաջինն է դուրս բերվում («pop» լինում)։ Որպես պատկերավոր օրինակ կարելի է դիտարկել ափսեների իրական հավաքածուն․ դուք սովորաբար նոր ափսեները ավելացնում եք վերևից, իսկ երբ հանում եք՝ վերցնում եք վերևի ափսեն առաջինը։

Հիմնական սկզբունք: Ստեկի վրա վերջին ավելացված տարրն առաջինն է հեռացվում։ Սա հայտնի է որպես LIFO (Last In, First Out) սկզբունք։

Օգտագործման օրինակներ

  • Զննարկչի պատմություն․ երբ սեղմում եք «Հետ» (Back) կոճակը, վերջին այցելած էջը հեռացվում է առաջինը։
  • Տեքստային խմբագրիչների «Չեղարկել» (Undo) ֆունկցիա․ ամենավերջին փոփոխությունն է չեղարկվում առաջինը, իսկ ավելի վաղ փոփոխությունները՝ հաջորդաբար։
  • Ծրագրավորման «Կանչերի ստեկ» (Call Stack)․ ֆունկցիաների կանչերը ավելացվում են ստեկի վերևից, և ամենավերջին կանչն է առաջինն ավարտվում։
  • Արտահայտությունների հաշվարկ․ օրինակ, մաթեմատիկական արտահայտությունների (հատկապես հետնամաս գրառման ձևով) հաշվարկման համար:
  • Ալգորիթմներում․ Դեպի խորություն որոնման (DFS) ալգորիթմում օգտագործվում է ուսումնասիրության ընթացքում հանգույցների հիշման համար:

Հիմնական գործողություններ

  • Push («ավելացնել»)․ Նոր տարրի ավելացում ստեկի վերևից։
  • Pop («հեռացնել»)․ Վերևի տարրի հեռացում ստեկից։
  • Peek («նայել վերևի տարրը»)․ Վերադարձնում է վերևի տարրը՝ առանց այն հեռացնելու։
  • Size («չափ»)․ Վերադարձնում է ստեկում պահվող տարրերի քանակը։
  • isEmpty («դատարկ է»)․ Ստուգում է, արդյոք ստեկը դատարկ է, թե ոչ։

Կոդային օրինակ

function InitializeStack():
    items = []     
    return items

function Push(stack, item):
    append item to stack
    return stack

function Pop(stack):
    if stack is empty then
        return "Stack is empty"
    else
        remove and return the last element from stack
    end if

function Peek(stack):
    if stack is empty then
        return "Stack is empty"
    else
        return the last element of stack
    end if

function Size(stack):
    return number of elements in stack

Թվային օրինակ

Սկսում ենք դատարկ ստեկից

[]

Տարրերի ավելացում (Push)

Ենթադրենք, հերթով ավելացնում ենք տարրերը՝ 5, 8 և 3.

Push(5) → [5]
Push(8) → [5, 8]
Push(3) → [5, 8, 3]   (Ստեկի վերևում այժմ 3-ն է)

Տարրի հեռացում (Pop)

Երբ հեռացնում ենք տարր, առաջինը դուրս է գալիս ամենավերջում ավելացված տարրը (3-ը)․

Pop() → 3 (Ստեկը դառնում է [5, 8])
Pop() → 8 (Ստեկը դառնում է [5])
Pop() → 5 (Ստեկը դառնում է [])
Pop() → "Stack is empty"

Ժամանակային բարդություն

  • Push («ավելացնել»)․ O(1) – տարրի ավելացումը կատարվում է հաստատուն ժամանակում։
  • Pop («հեռացնել»)․ O(1) – վերևի տարրի հեռացումը կատարվում է հաստատուն ժամանակում։
  • Peek («նայել»)․ O(1) – վերևի տարրին հասանելիությունն իրականացվում է հաստատուն ժամանակում։
  • isEmpty («դատարկ է»)․ O(1) – ստեկի դատարկ լինելու ստուգումը հաստատուն ժամանակում։
  • Size («չափ»)․ O(1) – տարրերի քանակի վերադարձը հաստատուն ժամանակ է, եթե այն հաշվարկվում է ընթացիկ ձևով։
